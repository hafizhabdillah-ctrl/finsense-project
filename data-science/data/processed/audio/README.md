# Audio Dataset (Processed)

Folder ini berisi dataset audio transaksi hasil preprocessing yang digunakan untuk mendukung pengembangan sistem pencatatan transaksi berbasis *voice input*.

## Struktur Folder

- `sample/`  
  Berisi beberapa contoh file audio hasil preprocessing.

## Download Full Dataset

## Audio Preprocessing

Seluruh audio hasil preprocessing telah melalui tahapan berikut:

- Mono conversion (`1 channel`)
- Resampling (`16 kHz`)
- Audio normalization
- Silence trimming

## Informasi Dataset

- Format file: `.wav`
- Audio channel: Mono (`1 channel`)
- Sample rate: `16 kHz`
- Bahasa: Bahasa Indonesia
- Jenis data: *Unstructured Data*
- Fungsi: Dataset audio untuk sistem transaksi berbasis suara
