# Audio Dataset (Raw)

Folder ini berisi dataset audio transaksi yang digunakan untuk mendukung pengembangan sistem pencatatan transaksi berbasis *voice input*.

## Struktur Folder

- `sample/`  
  Berisi beberapa contoh file audio mentah (*raw audio*).

## Download Full Dataset

Dataset audio lengkap tidak disertakan pada repository karena ukuran file yang besar.

Raw audio lengkap dapat diakses melalui Google Drive berikut:

👉 [Download Raw Audio Dataset](https://drive.google.com/file/d/1WOqGILXNeduF-t2Ejjq8Dp4evxrx39PH/view?usp=sharing)

## Informasi Dataset

- Format file: `.wav`
- Bahasa: Bahasa Indonesia
- Jenis data: *Unstructured Data*
- Fungsi: Input suara transaksi
